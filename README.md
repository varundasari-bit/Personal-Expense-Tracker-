💰 Personal Expense Tracker (Data Analysis Project)

📌 Overview

This project analyzes personal expenses to identify spending patterns and trends.

⚙️ Features

- Generates synthetic expense data
- Calculates total and category-wise spending
- Monthly trend analysis
- Visualizations (bar + line charts)

🛠 Tech Stack

- Python
- Pandas
- Matplotlib

▶️ How to Run

pip install pandas matplotlib
python expense_tracker.py

📊 Outputs

- "expenses.csv" → dataset
- "category_spending.png" → category analysis
- "monthly_trend.png" → trend analysis

🔍 Key Insights

- Identifies highest spending category
- Tracks monthly spending trends
- Calculates average expense
